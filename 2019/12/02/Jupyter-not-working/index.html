    Jupyter打不开
    2019/12/02 technique Python Jupyter Mac

    近日在mac中重装Anaconda，发现无法打开jupyter，总是报错。
    最终发现是因为语言的问题。
    在最近一版的Jupyter notebook中，菜单语言全都被改成了中文，这导致在系统打开jupyter的时候因为没有相应的中文支持而失败。

    解决方案：

    先打开 .bash_profile

    open ~/.bash_profile

    然后将以下内容添加到文件中

    export LC_ALL=en_US.UTF-8
export LANG=en_US.UTF-8

    然后重新载入

    source ~/.bash_profile

    重新运行jupyter notebook

    jupyter notebook

    Done! Enjoy!
